# 📊 Campus-wide Smart Utility Billing & Energy Dashboard

This repository contains the website developed as part of a capstone project focused on transforming raw electricity consumption data into an **interactive, decision-support dashboard**. Instead of relying on static reports, the platform enables stakeholders to **visualize, analyze, and optimize energy usage** across campus buildings.

---

## ⚙️ Tech Stack

- ⚛️ React – Interactive and responsive frontend  
- 🎨 Tailwind CSS – Clean and modern UI styling  
- 🔷 Lucide React – Consistent icon system  
- ☁️ Appwrite – Backend & database services  

---

## 💡 Project Overview

Traditional data reports are not sufficient for continuous monitoring and decision-making. This project bridges that gap by converting energy data into a **dynamic web interface**, allowing users to:

- Track monthly and annual electricity consumption  
- Analyze appliance-level energy usage (ACs, lights, fans, etc.)  
- Identify peak usage periods and seasonal trends  
- Receive actionable recommendations for energy savings  

---

## 🛠️ Development Approach

The development followed a structured pipeline:

- 🎨 Designed UI/UX using Figma prototypes  
- ⚛️ Developed frontend using React for scalability  
- 🎯 Focused on intuitive navigation and clean visualizations  

---

## 🌐 Website Structure & Features

### 📍 Home Dashboard
- Overview of total, average, peak, and lowest consumption  
- Monthly trends (line graph)  
- Appliance-wise breakdown (pie chart)  

---

### 📊 Analytics Dashboard
- Detailed time-series energy analysis  
- Appliance-wise consumption and cost trends  
- Seasonal variation insights (e.g., high AC usage in summers)  

---

### 🏢 Building Details
- Overview of GICT Building infrastructure  
- Floor-wise and space-wise breakdown  
- Area, layout, and facility insights  

---

### 💡 Recommendations Page
- Practical energy-saving suggestions  
- Appliance-specific optimization strategies  
- Encourages efficient energy usage behavior  

---

### 🔮 What-if Simulator
- Interactive decision-support tool for administrators  
- Modify inputs like HVAC load, lighting efficiency, etc.  
- Compare **actual vs projected consumption**  
- Helps evaluate impact of energy-saving strategies  

---

## ⚡ Non-Functional Requirements

- ✅ **Usability:** Simple and intuitive interface

## 🎯 System Outcomes

The dashboard serves as a centralized platform for energy monitoring and decision-making. It enables stakeholders to:

- Understand electricity consumption patterns  
- Identify high-energy usage areas  
- Make informed decisions to reduce energy waste  

👉 The system transforms raw data into **clear, actionable insights**, helping improve energy efficiency across campus.

## 🚀 Getting Started

### Clone the repository
```bash
git clone https://github.com/V1R3J/Energy-Dashboard.git
cd Energy-Dashboard
```
### Install dependencies
```bash 
npm install 
```
### Run the project 
```bash 
npm run dev
```
